## 支持的格式

Markdown 转 BBCode 支持：

- 标题
- 粗体、斜体、删除线、下划线
- 链接、图片
- 引用
- 无序列表、有序列表
- 行内代码、代码块
- 分割线
- 表格转为可读纯文本
- `<mask>` / `<spoiler>` 隐藏内容
- `<details><summary>标题</summary>内容</details>` 折叠内容
- 常见颜色、字号、对齐写法

BBCode 转 Markdown 支持：

- `[b]`、`[i]`、`[s]`、`[u]`
- `[url]`、`[img]`
- `[quote]`、`[code]`、`[mask]`
- `[color]`、`[size]`
- `[align]`、`[left]`、`[center]`、`[right]`
- `[list]`

遇到不认识的标签时，会尽量保留原文，避免误删内容。
